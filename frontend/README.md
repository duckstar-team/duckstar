# Duckstar Frontend

React 기반의 Duckstar 프론트엔드 애플리케이션입니다.

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm start
```

애플리케이션이 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 3. 빌드
```bash
npm run build
```

## 주요 기능

- 카카오 OAuth 로그인
- 애니메이션 투표 시스템
- 반응형 웹 디자인

## 기술 스택

- React 18
- React Router DOM
- Axios
- CSS3
